# bot-matrix

Учебная коллекция **стартеров (boilerplate-шаблонов) ботов** — для освоения разных
способов создания ботов разных мессенджеров в разных технологиях.

Идея — матрица **мессенджер × технология**: каждая ячейка — самостоятельный
минимальный, но запускаемый шаблон бота под конкретную пару.

## Раскладка

```
bot-matrix/
└── <мессенджер>/
    └── <технология>/   ← самостоятельный стартер
```

## Что такое «стартер» здесь

Минимальный набор: запускаемый echo-бот, пример конфигурации (`.env.example`),
